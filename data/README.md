# Data Folder

This folder will store data files for the future Web GIS project.

DATASETS USED

1. Count of Eviction Notices by Analysis Neighborhood and Year
Source: DataSF

This dataset contains eviction notice counts by San Francisco Analysis Neighborhood and year. The primary fields used in this project were neighborhood name, eviction notice count, and year. The dataset was used to measure spatial and temporal patterns of eviction notice activity across the city.

2. Analysis Neighborhoods
Source: DataSF

This dataset contains the polygon boundaries for San Francisco Analysis Neighborhoods. These neighborhood boundaries were created by grouping Census tracts into larger neighborhood units used for planning and analysis. The dataset was used as the primary spatial boundary layer for all maps and joins in the project.

3. San Francisco Land Use Dataset
Source: DataSF

This parcel-level dataset contains land use and residential unit information across San Francisco. The project used centroid coordinates, residential unit counts, and parcel geometry fields from the dataset. The data was used to calculate neighborhood-level residential unit totals and create a normalized eviction notice rate per 1,000 residential units.
