Project Overview

This project is part of my Web Mapping coursework. The goal is to build a basic website structure that can later support interactive maps, spatial data, and JavaScript-based web mapping tools. I am excited to learn how all of these systems work together in order for me to make my own website that is not locked behind a paid service. If you have any reccomendations on how to improve this site please feel free to reach out!
